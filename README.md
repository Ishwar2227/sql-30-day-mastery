
# 30-Day SQL Mastery — PostgreSQL

A structured 30-day SQL learning roadmap focused on data science 
and real-world application.

## Topics Covered
- Days 1-2: SELECT, WHERE, ORDER BY, Filtering
- Days 3-4: Aggregates, GROUP BY, HAVING
- Days 5-6: JOINs, Subqueries
- Days 7-8: CASE, CTEs, Window Functions
- Days 9-10: Query Optimization, Views, Stored Procedures
- Days 11-30: Advanced SQL (in progress)

## Environment
PostgreSQL 17 + pgAdmin 4

## Goal
Job-ready SQL for data science and analytics roles within 30 days.
